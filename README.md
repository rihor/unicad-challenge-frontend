# Delivery frontend code-challenge

Aplicação responsável por exibir as entregas e suas rotas e cadastrar novas entregas.

Endereço da aplicação: https://delivery-code-challenge-frontend.vercel.app/

## Como rodar

Crie um .env.local com as mesmos campos que o arquivo `.env.example`.

Para usar o Cypress você precisa rodar tanto a aplicação quanto o cypress.

### Tecnologias utilizadas

- Next.js (React Framework)
- Google Maps (@react-google-maps/api)
- Cypress (Testing Framework)
- Tailwind (CSS Framework)
- Yup
- Axios
- Dayjs
